# 函数

## 调用函数

* 要调用一个函数，需要知道函数的名称和参数，比如求绝对值的函数abs

	```
		>>> abs(100)
		100
		>>> abs(-20)
		20
		>>> abs(12.34)
		12.34
	```

* abs只有一个参数，参数类型如果不是数字会报错

* 而max函数max()可以接收任意多个参数，并返回最大的那个

* 数据类型转换, Python内置的常用函数还包括数据类型转换函数，比如int()函数可以把其他数据类型转换为整数

	```
		>>> int('123')
		123
		>>> int(12.34)
		12
		>>> float('12.34')
		12.34
		>>> str(1.23)
		'1.23'
		>>> str(100)
		'100'
		>>> bool(1)
		True
		>>> bool('')
		False
	```

* 也可以给函数起别名

	```
		>>> a = abs # 变量a指向abs函数
		>>> a(-1) # 所以也可以通过a调用abs函数
		1
	```

* hex()函数可以把一个整数转换成十六进制表示的字符串
